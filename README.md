# Legacy Shopping Portal Web Project

## Overview

This project is a **legacy static web-based shopping portal** developed using basic **HTML** and designed with **Microsoft FrontPage 4.0**. It demonstrates an early e-commerce style application structure with login, product browsing, and billing functionalities.

The system provides users with:

* A login interface
* A shopping dashboard using framesets
* Product catalogs for books and software
* A billing form for order summary

This project reflects foundational web development practices and serves as an academic or demonstration-level application.

---

## Project Objectives

* Simulate a simple online shopping experience
* Demonstrate static web page linking and navigation
* Organize products into categories
* Generate a billing form for customer orders

---

## Features

### 1. Login Page

* Collects **User ID** and **Password**
* Redirects users to the shopping dashboard after login
* Includes navigation to home and registration pages

**File:** `login.html`

---

### 2. Shopping Dashboard

* Uses **HTML framesets** for layout
* Includes:

  * Banner section
  * Navigation menu
  * Main content display area

**File:** `shopping.html`

---

### 3. Product Catalogs

#### Books Section

* Categorized into:

  * Computer Books
  * Fiction
  * General
* Displays Product ID, Product Name, and Price

**File:** `books.html`

#### Software Section

* Lists software-related products
* Includes pricing information

**File:** `software.html`

---

### 4. Billing Form

* Captures:

  * User ID
  * Order ID
  * Product details
  * Quantity
  * Date
  * Total cost
* Provides edit and cancel options

**File:** `bill.html`

---

## Strengths of the Project

* Simple and easy-to-understand structure
* Demonstrates early web application concepts
* Useful for learning HTML navigation and forms

---

## Limitations

* No backend/database integration
* No real authentication mechanism
* Static content only
* Uses outdated frameset design
* Limited responsiveness for modern devices
